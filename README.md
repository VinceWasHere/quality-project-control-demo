# Quality Project Control V2

## Funciones
- Login antes de la aplicación.
- Identificación automática del rol de cada usuario.
- Ejecución crea solicitudes y ve solo las propias.
- Calidad recibe, toma y evalúa inspecciones.
- Coordinación consulta usuarios y permisos.
- Gerencia y Presidencia tienen vistas de consulta.
- Datos persistentes en el navegador mediante localStorage.
- Publicable en GitHub Pages.
